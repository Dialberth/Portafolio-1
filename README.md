# Portafolio - Pagina donde aplico mi conocimiento en fronted para que no se me olvide

Este es un proyecto de práctica donde desarrollé un pagina, es algo sencillo, lo estare actualizando para que no se me olvide lo aprendido la practica hace al maestro **HTML5**.

### Características:
* Validación de campos (Nombre, Email, Mensaje).
* Lista desplegable de países.
* Imagenes.
* Estructura semántica para accesibilidad.
* ETC.
