# Algo por hacer — Proyecto Web de Práctica
Este proyecto es una página web sencilla que simula una pizzería, creada con el objetivo de practicar conceptos fundamentales de HTML, CSS.

La página incluye una sección de bienvenida, un catálogo de productos, una tabla de precios y stock, y un formulario básico de contacto.

### Características principales

Página de inicio con mensaje de bienvenida
Imagen destacada (vista de Francia desde el río Seine)
Catálogo de pizzas con precios y disponibilidad
Formulario de contacto con validación mínima
Pie de página con derechos reservados

### Aprendizajes aplicados

Durante este proyecto puse en práctica:
Creación y manejo de repositorios en GitHub
Uso de commits descriptivos
Creación y manejo de ramas
Edición desde ramas y fusión de cambios
Comandos básicos de Git
Organización de archivos HTML y estructura semántica

### Tecnologías utilizadas
HTML5
CSS3
